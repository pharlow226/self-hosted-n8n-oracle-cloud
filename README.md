
# Self-Hosted n8n on Oracle Cloud — Free Forever

A complete guide and reference for deploying a production-ready n8n workflow automation instance on Oracle Cloud Infrastructure using Docker and Coolify — at zero cost.

This repository documents the full setup across four scenarios, so anyone can follow along regardless of their budget or whether they own a domain.

---

## What This Covers

- Provisioning a free Oracle Cloud ARM or AMD VM
- Upgrading to Pay As You Go (and why it's still free)
- Installing and configuring Coolify as a container management platform
- Deploying n8n with PostgreSQL via Coolify
- Setting up a custom domain with automatic HTTPS
- Alternatives for people without a domain or without a PAYG upgrade
- Webhook URL options for every scenario (Cloudflare Tunnel, ngrok, sslip.io, Caddy)

## Security Notes

- SSH key authentication only — password authentication is disabled by default on Oracle Cloud VMs
- Oracle Security List controls inbound traffic at the network level
- Ubuntu iptables rules provide a second layer of firewall protection
- All sensitive values (passwords, API keys, tokens) are managed via Coolify environment variables — never hardcoded
- Let's Encrypt SSL certificates are provisioned and renewed automatically
- n8n runs as a non-root user inside the container

---

## Environment Variables

Never commit real credentials. All sensitive values should be stored in Coolify's environment variable manager or as shell environment variables. The following are required:

```env
# n8n
WEBHOOK_URL=https://your-domain.com
N8N_EDITOR_BASE_URL=https://your-domain.com
N8N_HOST=your-domain.com
N8N_PROTOCOL=https
N8N_SECURE_COOKIE=true
GENERIC_TIMEZONE=Europe/Berlin
TZ=Europe/Berlin

# PostgreSQL (auto-generated by Coolify — do not hardcode)
POSTGRES_USER=
POSTGRES_PASSWORD=
POSTGRES_DB=n8n
```

## Webhook Setup

For external services (WhatsApp, Stripe, GitHub, Lovable) to reach your n8n workflows, you need a public HTTPS URL.

| Setup | URL Format | Cost |
|---|---|---|
| Custom domain + Coolify | `https://n8n.yourdomain.com/webhook/...` | Domain ~$1–10/year |
| sslip.io + Coolify | `https://YOUR_IP.sslip.io/webhook/...` | Free |
| Cloudflare Tunnel | `https://random.trycloudflare.com/webhook/...` | Free |
| ngrok static domain | `https://yourname.ngrok-free.app/webhook/...` | Free |

> After changing your `WEBHOOK_URL`, always deactivate and reactivate all existing workflows in the n8n editor to re-register the new URL.
